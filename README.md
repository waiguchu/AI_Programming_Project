## Student performance prediction using machine learning
this project uses machine learning algorithms to predict student academic performance based on factors such as study hours,attendance,past grades and demographic information.The dataset is preprocessed and analysed using phython libraries such as pandas and NumPy and a classification or regression moddel is trained using scikit-learn to predeict whether a student is likel to pass or fail.The goal is to help educators identify at-risk students early and improve learning outcomes.

## Python libraries used
-numpy
-pandas
-scikit-learn
